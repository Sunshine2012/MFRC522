#RFID读卡器

##介绍

今天要教大家做的东西是一个**RFID读卡器**,程序已经**预先烧入单片机**中,同学们只需正确接线,即可利用**几行简单的代码**读取RFID卡片.

##原料

* Arduino Nano
* MFRC522 RFID模块
* 蜂鸣器
* 导线
* USB连接线

##制作过程

###RC522

首先将**RC522**与**Arduino**连接起来,对应关系如下表:

| RC522 | Arduino |
|------:|:--------|
| SDA	| D9	|
| SCK	| D13	|
| MOSI	| D11	|
| MISO	| D12	|
| GND	| GND	|
| RST	| D5	|
| 3.3V	| 3V3	|

###蜂鸣器

然后将**蜂鸣器**接到单片机上,对应关系如下表:

| 蜂鸣器 | Arduino |
|------:|:--------|
| 长的脚	| D6	|
| 短的脚	| GND	|

* 注意:单片机有两个GND,任意接一个都行.

##连接电脑

做完了上面两步之后,就可以将USB线接入电脑了.
